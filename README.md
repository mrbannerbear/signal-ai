feedback-triage-ai is a full-stack Next.js + Supabase web app that helps teams quickly process customer feedback. Users can submit feedback, filter by urgency or category, and leverage a simple AI tool to summarize and prioritize issues. Built with modern tooling (Next.js 16, Supabase, Vercel AI SDK, Tailwind/Shadcn UI) to showcase full-stack skills and practical AI integration.
